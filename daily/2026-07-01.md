### 今日解决疑问:

为什么stdio.h中存在`#define stdin stdin`这样的语句?

>这是出于规范的写法, 全代码如下. `C89/C99 say they're macros.  Make them happy.`,  这里实际上最重要的是前面的extern, 然后出于规范要求就写成了宏. 

```c
/* Standard streams.  */
extern FILE *stdin;		/* Standard input stream.  */
extern FILE *stdout;		/* Standard output stream.  */
extern FILE *stderr;		/* Standard error output stream.  */
/* C89/C99 say they're macros.  Make them happy.  */
#define stdin stdin
#define stdout stdout
#define stderr stderr
```

为什么printf后面通常会加一个`\n`?

>出了方便阅览外, 由于printf是按行缓存, 加了`\n`之后会直接flush, 这样会更稳输出.
