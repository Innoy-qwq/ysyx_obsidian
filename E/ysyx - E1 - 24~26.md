2、编一个函数，输入一个字符串，要求做一个新字符串，把其中所有的一个或多个连续的空白字符都压缩为一个空格。这里所说的空白包括空格、'\t'、'\n'、'\r'。例如原来的字符串是：

```
This Content hoho       is ok
        ok?

        file system
uttered words   ok ok      ?
end.
```
压缩了空白之后就是：

```
This Content hoho is ok ok? file system uttered words ok ok ? end.
```

实现该功能的函数接口要求符合下述规范：

```
char *shrink_space(char *dest, const char *src, size_t n);
```

各项参数和返回值的含义和`strncpy`类似。完成之后，为自己实现的函数写一个Man Page。

>答: 代码如下:

```c
#include <stdio.h>

// 函数原型应当是char *shrink_space(char *dest, const char *src, size_t n);
// 实现的效果是将src的字符串选取n个, 进行空格压缩并填入dest, 若字符串不满n个则后续填\0

char *shrink_space(char *dest, const char *src, size_t n)
{
    char *ret = dest;
    int flag = 0; // 表示是否已经出现空位;
    for (int count = 0; count < n; count++)
    {
        if (*src == ' '  ||
            *src == '\t' ||
            *src == '\n' ||
            *src == '\r'){
            if(!flag){
            *dest = ' ';
            dest++;
            flag = 1;}
            }
        else{
            *dest = *src;
            dest++;
            flag = 0;
            }

        if (*src != '\0')
            src++;
    }
    
    return ret;
}

int main(void)
{
    char a[30];
    printf("%s", shrink_space(a, "Hello \n \t Innoy", 15));
};
```

### malloc and free

malloc申请内存, 返回void* 类型指针, 

>注意: malloc申请的内存被free掉, 只是这一块地址不再归属于用户程序, 指向这块地址的指针当然仍是存在的, 这时我们最好手动把这个野指针指向NULL来保证安全.

### 传入参数与传出参数

传入参数就是作为指针参数进入函数后,读取其数据进行使用的参数. 传出函数就是作为指针参数进入函数后, 被写入数据用于外部读取的参数, value-result函数就是既作为传入读取也作为传出调用的指针.

>传出参数和result-value参数作为函数接口完全相同.


两层指针的用法应当很明白, 没什么额外的信息. 

### 回调函数

经典的回调函数, 函数指针作为参数被传入到函数中提供调用, 一般使用viod*.
C++的泛型算法（Generics Algorithm）中可以使用回调函数, 其作用是可以有一个灵活的比较函数, 外部调用者自行实现两个对象的比较函数, 然后将这个比较函数的函数指针作为回调函数传入就可以了. 这样的话可以实现比如结构体之间的比较. 

这里效果上就有点类似于cpp的运算符重载

int cmp(const void *a, const void *b){
    return strcmp((const char *)a, (const char *)b);

/*尝试使用泛型的二分查找, 这里求助了AI作为参考, 返回这里返回了字符串的地址, 修改前面的cmp可以变成查找int, 返回int的地址, 总之返回的是数组中的地址位置*/
void *binary_search(
    void *base,
    size_t n,
    size_t size,
    void *key,
    int (*cmp)(const void *, const void *))
{
    char *arr = base;
    size_t left = 0, right = n;

    while (left < right)
    {
        size_t mid = (left + right) / 2;
        void *elem = arr + mid * size;

        int c = cmp(key, elem);

        if (c == 0)
            return elem;
        else if (c < 0)
            right = mid;
        else
            left = mid + 1;
    }

    return NULL;
}

int main(){
    char exp[5][8] = {
        "inooy",
        "abc",
        "that's",
        "bbbbb",
        "aocccc"};
    printf("%s\n", (char *) binary_search(exp, 5, 8, "abc", cmp));

> 泛型的三种:
> 
>1. 指针数组模型, 也就是K&R中给出的类型, 用指针的话, 数组中可以指向任何类型, 这样的话就形成了一种泛型, 优点是实现简单, 缺点是不够抽象.
>2. void* + size模型, 这种模型是c标准库的实现, 访问方法基于地址偏移, 也就是`base + i * size`. 优点是足够抽象和泛型.
>3. template, 也就是cpp的模板, 在函数前面加template结构直接可以做自动类型替换, 属于不用人工写的现成抽象.
